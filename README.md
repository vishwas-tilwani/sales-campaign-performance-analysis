# Sales Campaign Performance Analysis (Excel Dashboard)

## Overview
This project analyses campaign performance using Microsoft Excel.  
The goal was to build an interactive dashboard to evaluate marketing and sales performance across sectors, audiences, and company sizes.

## Business Questions
- Which sectors generate the highest Return on Ad Spend (ROAS)?
- Which audience groups convert the best?
- Which sectors have the highest Cost Per Acquisition?
- How effectively are campaign budgets being utilised?
- How has campaign activity changed over time?

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design

## Key Metrics Analysed
- Revenue
- Conversion Rate
- Click Through Rate
- Cost per Click
- Cost per Acquisition
- Budget Utilisation
- ROAS

## Files in this Repository
- `cleaned dataset.xlsx` – cleaned campaign dataset
- `Sales Operations Commercial Insight.xlsx` – Excel dashboard
- `Sales Operations Commercial Insight - Presentation.pptx` – summary of insights and findings
